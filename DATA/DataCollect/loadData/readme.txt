It is used to load kinematic and sEMG data from raw files.
main.m      collect kinematic and sEMG data of everyone.
loadEmg.m	load sEMG matrix from ASC file.
loadQuat.m	load quaternions from Quadx.txt
readRes.m	load segment index and motion from Result.txt
restruct.m	A script for restruct dataset, affect(i).data to affect.data(i)
reEMG.m     restruct datasets contain kinematic and sEMG data.
syndata.m   restruct synchronous data from raw dataset
delEmpt.m   Delet the empty values in restruct data.
